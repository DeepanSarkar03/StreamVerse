# **App Name**: StreamVerse

## Core Features:

- Video Listing: Fetch and merge video file lists from Google Drive and OneDrive, generating playable URLs.
- Video Streaming: Stream videos directly in the browser using the HTML5 video tag.
- Video Upload (Google Drive): Upload video files to a specified Google Drive folder.
- Video Upload (OneDrive): Upload video files to a specified OneDrive folder.
- Client-Side Search: Enable searching for videos directly in the browser.
- Thumbnail Display: Display video thumbnails in a grid layout.

## Style Guidelines:

- Primary color: Dark gray (#222222) to mimic Netflix's dark interface.
- Background color: Very dark gray (#0A0A0A), nearly black, to enhance contrast with video content.
- Accent color: Red (#E50914) for interactive elements, calls to action, and highlights.
- Body and headline font: 'Inter', a sans-serif font, for clear and modern readability.
- Grid layout for video thumbnails, similar to Netflix's browsing experience.
- Simple, clear icons for upload and search functionalities.
- Subtle hover animations on video thumbnails for a responsive feel.
